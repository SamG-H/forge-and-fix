# Forge And Fix
### A place for people to share ideas and work on projects together.
Sign up for an account, create projects, add issues(bugs/features), and assign tasks to yourself.
## Installation
    1. Clone this repo
    2. $ bundle install
    3. $ rails s
## Usage
    Go to localhost:3000/ in your brower
